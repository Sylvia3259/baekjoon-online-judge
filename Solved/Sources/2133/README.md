# 타일 채우기 [🔗](https://www.acmicpc.net/problem/2133)

## 문제
<p>3×N 크기의 벽을 2×1, 1×2 크기의 타일로 채우는 경우의 수를 구해보자.</p>

## 입력
<p>첫째 줄에 N(1 ≤ N ≤ 30)이 주어진다.</p>

## 출력
<p>첫째 줄에 경우의 수를 출력한다.</p>
