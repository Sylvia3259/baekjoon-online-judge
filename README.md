## 참고사항
* 본 프로젝트에는 백준 문제 풀이를 편하게 할 수 있도록 도와주는 ```BOJ Helper``` 서비스가 포함되어 있습니다.
  * 아래의 기능을 지원합니다.
    - 자동 채점
    - 소스코드 백업
    - 마크다운 형식 문제 설명 생성
* 사용은 자유이나 기본적으로 제가 사용하는 환경에 맞게 제작되었습니다.
  * 아래 환경을 기준으로 제작되었습니다.
    - Windows 10
    - Visual Studio 2019
    - MinGW W64
    - C++17
* 만약 본인의 환경에 맞춰 사용하시려면 세부적인 부분들은 직접 수정하셔야 합니다.
* 아래 패키지(또는 모듈)들이 필요합니다.
  - Requests
  - BeautifulSoup4
  - pandas
  - pyperclip
